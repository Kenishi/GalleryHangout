Privacy Policy

What data is collected?
The only data that gets saved is image data which is uploaded to the Imgur website. As such, the user should review the Imgur Privacy Policy as well. This image data is not transmitted to any server after posting on Imgur. Realize that all image data uploaded using Gallery Hangout app can be viewed publicly by anyone should they have the URL to the image.

In addition to Imgur's privacy policy, the user should also review the Google+ Privacy Policy to see what potential data may be collected by Google.
